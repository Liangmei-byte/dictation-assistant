# 小学生听写助手

## 项目简介

小学生听写助手是一款专为小学生设计的在线听写工具，旨在帮助父母没时间辅导作业的小学生独立完成听写任务。通过提供词语输入、随机朗读、计时提醒等功能，让小学生能够自主完成听写练习，提高学习效率。

## 核心功能

- **词语输入**：支持用户输入要听写的词语，用空格分隔
- **词语打乱**：自动打乱词语顺序，确保随机不重复读取
- **语音朗读**：使用浏览器内置语音功能朗读词语，支持柔和有感情的声线
- **计时提醒**：每个词语间设置10秒时间间隔，到时间自动提醒
- **词语隐藏**：听写开始后隐藏所有词语和输入模块，防止学生查看
- **词语显示**：听写完成后重新显示所有词语，方便学生批改
- **历史记录**：保存用户听写历史，支持查看过往记录
- **准备时间**：听写开始前提供5秒准备时间
- **用户认证**：支持用户注册、登录、登出功能
- **数据存储**：登录用户的听写历史存储到服务器，未登录用户存储到本地
- **响应式设计**：适配不同设备屏幕

## 技术实现

- **前端**：HTML5, CSS3, JavaScript
- **语音合成**：使用浏览器内置的Speech Synthesis API
- **数据存储**：
  - 未登录用户：使用浏览器的localStorage
  - 登录用户：使用MongoDB数据库
- **后端**：Node.js, Express, MongoDB, JWT
- **响应式设计**：使用CSS媒体查询

## 如何使用

### **1. 环境准备**
- **Node.js**：后端服务器运行环境，可从[官网](https://nodejs.org/)下载并安装
- **MongoDB**：数据库，用于存储用户信息和听写历史，可从[官网](https://www.mongodb.com/)下载并安装

### **2. 启动后端服务器**
1. 进入backend目录：
   ```bash
   cd d:\code\python\Dictation\backend
   ```
2. 安装依赖：
   ```bash
   npm install
   ```
3. 启动MongoDB服务
4. 启动后端服务器：
   ```bash
   npm run dev
   ```
5. 服务器会在 `http://localhost:5000` 上运行

### **3. 前端使用**
1. **本地运行**：
   - 找到 `index.html` 文件
   - 双击打开它（会在默认浏览器中运行）

2. **使用步骤**：
   - 新用户点击"注册"按钮创建账号
   - 已有用户点击"登录"按钮登录
   - 登录后，在文本框中输入要听写的词语，用空格分隔
   - 点击"开始听写"按钮
   - 系统播报"5秒钟后开始听写"并显示5秒倒计时
   - 5秒后，系统会隐藏输入模块和词语列表，开始朗读第一个词语
   - 10秒后自动朗读下一个词语
   - 所有词语朗读完毕后，会显示"听写完成"并重新显示输入模块和词语列表
   - 点击"历史记录"按钮可以查看过往听写记录
   - 点击"登出"按钮可以退出账号
   - 可以随时点击"停止"按钮暂停，或"重置"按钮重新开始
